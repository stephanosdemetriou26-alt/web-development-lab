# Web Development Projects

Αυτό το αποθετήριο περιέχει εργασίες και παραδείγματα κώδικα για το μάθημα της ανάπτυξης ιστού.

## Περιεχόμενα
Το repository περιλαμβάνει δύο βασικές ενότητες:

### 1. Σύστημα Διαχείρισης Χρηστών (PHP & MySQL)
Ένα σύστημα για τη διαχείριση εγγραφών σε βάση δεδομένων.
* **Δημιουργία Πίνακα:** Αυτόματη δημιουργία πίνακα `iisXXXX` στη βάση δεδομένων[cite: 3].
* **Εισαγωγή Δεδομένων:** Φόρμα για προσθήκη email με έλεγχο διπλοτύπων[cite: 1].
* **Διαγραφή:** Δυνατότητα διαγραφής εγγραφών με βάση το ID[cite: 4].

### 2. Εφαρμογή Text Resizer (JavaScript)
Εφαρμογή που επιτρέπει στον χρήστη να αλλάζει το μέγεθος του κειμένου.
* Αυξομείωση μεγέθους κατά 5px με κουμπιά[cite: 2].
* Όρια μεγέθους γραμματοσειράς από 10px έως 50px[cite: 2].

## Τεχνολογίες
* **HTML5**: Δομή σελίδων και φόρμες[cite: 1, 3].
* **CSS3**: Μορφοποίηση χρωμάτων και στοίχιση[cite: 1, 2].
* **JavaScript**: Δυναμική αλλαγή μεγέθους κειμένου[cite: 2].
* **PHP**: Server-side λογική και σύνδεση με βάση[cite: 1, 4].
* **MySQL**: Αποθήκευση δεδομένων[cite: 3].
